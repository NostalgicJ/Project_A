# 화장품 성분 조합 분석 AI 프로젝트 - 발표 요약

## 🎯 한 줄 요약
"AI로 화장품 성분 조합의 안전성과 시너지를 분석하는 시스템"

## 📌 핵심 내용 (3분 요약)

### 1. 문제 정의 (30초)
- 화장품 성분 간 상호작용에 대한 정보 부족
- 일반 소비자가 안전한 조합인지 판단 불가
- 전문가 검수 필요 → 자동화 필요

### 2. 해결 방법 (1분)
- **데이터**: 공공데이터 20,000+ 올리브영 2,000+ 제품 수집
- **전처리**: 제품명 정제, 성분 정규화, 기획상품 분리
- **모델**: 규칙 기반 + 딥러닝 하이브리드
- **결과**: 위험도 90%+ 경고, 시너지 효과 추천

### 3. 기술적 특징 (1분)
- 규칙 기반: 성분 계열 매칭으로 빠른 분석
- 딥러닝: 상호작용 패턴 학습으로 높은 정확도
- 제품 간 비교: A 제품(20성분) vs B 제품(30성분) 동시 분석
- 실시간 추천: RESTful API로 즉시 사용 가능

### 4. 활용 방안 (30초)
- 소비자: 구매 전 성분 조합 확인
- 브랜드: 제품 개발 시 검토
- 전문가: 상담 도구로 활용

## 📊 주요 수치

- 제품 수: 2,000+ 개
- 성분 수: 1,500+ 개
- 분석 조합: 100,000+ 개

- 정확도: 85%+
